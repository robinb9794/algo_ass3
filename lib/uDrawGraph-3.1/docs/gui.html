On this page, the default menus are displayed. An application connected to the API is able to reconfigure the whole user interface of uDraw(Graph), especially the content and order of menus. Refer to user interface configuration for details. Thus the actual menus may be different if you are using uDraw(Graph) together with an application program.

All screenshots shown in this section are taken from the Linux® version running on SuSE® 9.2 with the KDE 3.3 window manager using the "SUSE" window decoration style. On a different environment, the texts, layouts and window borders may vary. Some dialogs like the file browser may even be completely different, for example on the Windows® or on the Mac OS® X platform where the native file browser is used for convenience.
