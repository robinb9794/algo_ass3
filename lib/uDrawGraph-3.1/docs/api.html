Application Programming Interface Reference

One of the fundamental concepts of uDraw(Graph) is to provide a generic user interface for graphs that can be used on top of an arbitrary application program. The application is able to access nearly all operations of uDraw(Graph) by using the API (Application Programming Interface) described here. So you do not have to "reinvent the wheel" if you are a programmer of an application looking for a powerful graphical user interface to display the application's graph data structures. Simply plug uDraw(Graph) with your application program and use the API to get state-of-the-art graph visualization technology.
