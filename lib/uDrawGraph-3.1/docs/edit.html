Graph Editor Application

This chapter covers the interactive graph editor application that is delivered with uDraw(Graph). The graph editor can be connected to the uDraw(Graph) API. It is an example application to demonstrate the features of the API and it is a very useful and comfortable tool to create graphs very easily.
