Naming Scheme Window

This graph editor dialog window is used to specify the naming scheme applied for newly inserted nodes. Nodes are inserted into the graph with menu Edit/Node/Insert Root or Edit/Node/Insert Child, by using graph editor icons or with drag and drop. Each new node automatically gets the text from the naming scheme by replacing the placeholder character # with the current unique node number which is incremented with each new node.

Dialog Elements of the Naming Scheme Window

  • Node Text
    Here you can specify the general text pattern used for the text of newly inserted nodes. It is possible to use multi-line text by typing RETURN. The special character "#" is a placeholder for the current node number.
  • Apply
    Push button to accept the current settings of this dialog window. Without pressing this button, modifications to these options are not considered.
  • Cancel
    Push button to close the naming scheme dialog window without performing any action.
  • Help
    Push button to start the online help system with this page.
