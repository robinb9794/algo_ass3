New Features in uDraw(Graph) 3.1

Starting with uDraw(Graph) 3.1 our product has a new name. While there is only a small list of new features, most changes have been forced by the renaming. These changes are described in the backward compatibility section.

  • New license conditions
  • New arrow head
  • Text of unselected edges in black instead of grey
  • Updated Tcl/Tk version to 8.4.9

New Features in version 3.0

Version 3.0 has been a major release with a lot of significant new features. For users familiar with previous versions, all modifications are described in this chapter.

  • Installation and Start Dialogs
  • Revised User Interface, Configuration and Internationalization
  • New Rendering Engine with more Comfort and Performance
  • Edge Labels, Splines and Tooltips
  • Color Icons and new Edge Options
  • New Printing Features and Export
  • Improved Graph Editor
  • Miscellaneous Goodies
  • Application Programming Interface via Sockets
  • New API Commands and Features
  • New Command-Line Options
  • New Platform Microsoft® Windows®
  • Removed Features and Important Modifications
